# Programmable-Power-Supply
这是一个用C#编程语言￼写的PC端的用户界面，来控制可编程电源。主要是用来控制固纬电子有限公司的可编程直流电源(PSW 系列， 如PSW 30-180），也可以控制其他一些产品基于IEEE488.1的标准接口。
